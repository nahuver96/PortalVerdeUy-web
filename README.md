Portal Verde fue creado con la intención de facilitar el acceso a farmacias habilitadas por el IRCCA (Instituto de Regulación y Control del Cannabis) y a Clubes Cannabicos del Uruguay. Surgió a partir de una necesidad común de la comunidad de tener un medio ágil para acceder a dicha información y no depender de exhaustas búsquedas online. De un bro, para bros✌️

Si te parece útil la app y te gustaría colaborar para que siga recibiendo actualizaciones y esté al día con la información, puedes aportar para la causa en el siguiente link <3.
